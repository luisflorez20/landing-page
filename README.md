# Landing Page de Ventas

Estructura inicial para una landing page de ventas estática.

## Arquitectura de carpetas

- `public/`
  - `index.html`: página principal de la landing page.
- `src/`
  - `assets/`: recursos como imágenes e íconos.
  - `styles/`: hojas de estilo CSS.
  - `scripts/`: JavaScript de interacción.

## Cómo usar

1. Ejecuta `npm install` en la raíz del proyecto.
2. Inicia el servidor de desarrollo con `npm run dev`.
3. Abre la URL que muestre Vite en la terminal.
4. Modifica `src/styles/main.css` para ajustar el diseño.
5. Cambia contenido en `index.html` y la lógica en `src/scripts/main.js`.
